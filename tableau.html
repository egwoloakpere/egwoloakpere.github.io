Parameters in Tableau

Parameters act as versatile placeholders that users can adjust to impact visualizations and calculations, offering an elevated level of flexibility and user interactivity. Parameters extend the interactivity of a dashboard by allowing users to control values such as thresholds, time periods, target metrics, and categorical options. This flexibility enables users to create dynamic comparisons, switch between measures, and fine-tune what data they see without needing to adjust underlying data sources or develop a new visualization.

For example, with parameters, users can toggle between different Metrics/Measures (like Sales, Profit, Quantity, etc.) and Dimensions (like Country, City, Product_category, etc.) within a single visualization.
